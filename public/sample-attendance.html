Sample Excel Format for Salary Processor

This document explains the expected format for the Excel file used with the Salary Processor application.

Important: Your Excel file must match this exact format to be processed correctly.

Excel Structure

The Excel file should contain attendance data with the following columns:

Column Description Format Example
Employee ID Unique identifier for the employee String (must match IDs in system) EMP001
Date Date of attendance record Date format (MM/DD/YYYY) 06/01/2025
In Time Time employee arrived Time format (HH:MM) 09:00
Out Time Time employee left Time format (HH:MM) 17:00
Status Attendance status code String (P=Present, A=Absent, WO=Week Off, WOP=Week Off Present) P
Hours Worked Total hours worked (optional) Numeric 8.0

Sample Data

Here's a sample of what your Excel data should look like:

Employee ID Date In Time Out Time Status Hours Worked
EMP001 06/01/2025 09:00 17:00 P 8.0
EMP001 06/02/2025 09:15 17:30 P 8.25
EMP001 06/03/2025 A 0.0
EMP001 06/04/2025 08:45 17:15 P 8.5
EMP002 06/01/2025 09:30 18:00 P 8.5

Important Notes

  • The first row of your Excel file should be a header row with these column names.
  • Employee IDs must match the IDs in the system (EMP001, EMP002, EMP003, EMP004, EMP005).
  • Late arrivals (after 10:00 AM) will be automatically flagged as late and may affect salary calculations.
  • The system calculates salary based on attendance, worked hours, and late marks.
